# The UX Review Blog

A modern responsive blog website built using HTML and CSS.

---

## 🚀 About the Project
This project is a practice blog UI with a bold brutalist design style.  
It focuses on strong typography, card layouts, and responsive design.

---

## ✨ Features
- Fully responsive design
- Hero section with bold typography
- Latest articles cards layout
- Sidebar with trending posts & categories
- Authors section
- Community testimonials section
- Footer with newsletter form

---

## 🛠️ Built With
- HTML5
- CSS3
- Google Fonts (Exo)

---

## 📱 Responsive Design
Works on:

- Mobile
